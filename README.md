# Cannon Defence

A tower defense game built with Godot 4.2.

**Play now:** https://krupnikas.github.io/cannonDefence/

## Features

- 9 unique cannon types with special abilities
- 6 enemy types with different behaviors
- 15 levels with 3-star rating system
- Status effects (burn, freeze)
- Special mechanics (critical hits, dodge, splash, pierce, chain lightning)
- A* pathfinding
- Save/load progress

## Controls

- **Click** - Place cannon
- **Right-click** - Sell cannon
- **1-9** - Select cannon type
- **P / ESC** - Pause game
- **Mouse wheel** - Zoom
- **Middle mouse** - Pan camera
